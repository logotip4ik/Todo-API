# Todo Api
Todo api with Django and html frontend

## Installation

* Install dependencies:
    ```bash
    pip install django django-restframework
    ```
* Clone repo:
    ```bash
    git clone https://github.com/logotip4ik/Todo-API.git
    ```

## Usage
Just start the server:
```bash
python manage.py runserver
```
